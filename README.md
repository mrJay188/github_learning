# GitHub Lab Practical

## Aim

To perform basic GitHub operations using **only the GitHub website**, including creating repositories, branches, files, pull requests, issues, and collaboration.

---

## Tools Used

* Web Browser
* GitHub (github.com)

---

## Practical Tasks Performed

### 1. Repository Creation

* Created a public repository named **github-lab-practical**
* Initialized the repository with a README file

---

### 2. Branching and File Creation

* Created three branches: `branch1`, `branch2`, and `branch3`
* Added files in each branch:

  * **file1.txt** → GitHub basics
  * **file2.txt** → Git branching
  * **file3.txt** → GitHub collaboration
* Committed changes separately in each branch

---

### 3. Pull Requests and Merging

* Created pull requests from each branch to `main`
* Successfully merged all branches into the main branch

---

### 4. Issue Management

* Raised an issue titled **“Update README file”**
* Updated README.md with additional details
* Closed the issue after resolving it

---

### 5. Collaboration

* Added a collaborator to the repository
* Collaborator edited README.md
* Changes were merged using a pull request

---

## Final Output

The main branch contains:

* README.md
* file1.txt
* file2.txt
* file3.txt
